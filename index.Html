Cliente/Servidor

Gracias a la arquitectura Cliente/Servidor hemos logrado desarrollar paginas web con mayor interacción entre el Cliente y el servidor, ya sea los diferentes navegadores que se ofrecen hoy en día y los servidores con diferentes funciones y mayor lectura de lenguajes de programación, por lo que aumenta el uso de diferentes herramientas, esto ha favorecido a los desarrolladores a elaborar paginas web con mayor conexión, comunicación y mayor capacidad de datos, por lo que esta arquitectura es una herramienta muy utilizada por los desarrolladores. Los usuarios buscan hoy en día sitios web con mayor interacción y herramientas disponibles, con diferentes componentes multimedia por lo que la arquitectura Cliente/Servidor ha sido un aliado y una manera más eficiente para elaborar sitos web dinámicos entre el Cliente y el servidor, ya que esta mantiene una relación independiente, entre la lógica de presentación que se puede referir al FrontEnd y la lógica de negocio que puede ser las acciones que se esperan a cambio, solicitando información a la lógica de datos donde esto vuelve al usuario respectivamente, también conocida como Backend. Muchos más usuarios ven los beneficios que esta comunicación tiene para ellos, ya sea mayor seguridad en los datos, ya que contamos con firewall donde solo los servidores pueden acceder a los datos que se están solicitando, mejor comunicación en la red y actualizaciones.

Analisis de Requisitos y su importancia

Es fundamental el análisis de requisitos en un proyecto de desarrollo web, ya que por medio de este podemos desarrollar las aplicaciones supliendo las expectativas y necesidades que los clientes presenten, logrando así cumplir con éxito esas peticiones. Estos requisitos se pueden obtener por medio de diferentes herramientas de recolección de datos ya sean; entrevistas, apoyo técnico, formularios, encuestas, cada una de estas opciones tienen valores cualitativos como cuantitativos, en esta etapa es necesario que el desarrollador se vuelva cliente y el cliente se vuelva desarrollador, con esta garantía podemos entender que estamos construyendo y que vamos a utilizar.
